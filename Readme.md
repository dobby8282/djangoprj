### Django 프로젝트

- django 설치
```
pip install django
```

- django 프로젝트 생성 및 config 생성
```
django-admin startproject config .
```

- django App 생성
```
python manage.py startapp 앱이름

python manage.py startapp iris
```


- django 실행
```
python manage.py runserver
```

- 마이그레이션 적용
```
python manage.py migrate
```

- 마이그레이션 변경파일 생성
```
python manage.py makemigrations
```
